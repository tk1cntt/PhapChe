    - Replace `<PageHeader title="Audit" description="..." />` with:
      ```tsx
      <Flex vertical gap={4} style={{ marginBottom: 16 }}>
        <Typography.Title level={3} style={{ margin: 0, fontSize: 30, fontWeight: 600 }}>

        </Typography.Title>

        </Typography.Paragraph>
      </Flex>
      ```
    - Keep `<Card>` as antd Card (same component name from antd).
    - Replace `<Table headers={[...]}>...table rows...</Table>` with antd `<Table>`:
      ```tsx
      <Table
        dataSource={auditEvents}
        rowKey="id"
        columns={[
          { title: 'Thoi gian', dataIndex: 'createdAt', key: 'createdAt',
            render: (val: Date) => new Intl.DateTimeFormat('vi-VN', { dateStyle: 'medium', timeStyle: 'short' }).format(val),
            width: 180 },
          { title: 'Actor', key: 'actor',
            render: (_: unknown, record: typeof auditEvents[0]) => record.actor?.email ?? 'system',
            width: 200 },
          // ... rest of columns
        ]}
        pagination={false}
        size="middle"
        bordered
      />
      ```
    **Tag (Badge) replacement:**
    - Replace `<Badge tone="info">{event.action}</Badge>` with `<Tag color="blue">{event.action}</Tag>`
    - The tone-to-color mapping per UI-SPEC: neutral='default', info='blue', warning='orange', accent='cyan', destructive='red', outline='default'

    **Empty state:** Replace the `<tr><td colSpan={7}>...` pattern with antd Table's `locale={{ emptyText: 'Chua co su kien kiem toan nao.' }}`

    **Type safety:** Define column types inline with the table. Use `import type { AuditEvent } from ...` if a specific type exists, or use inline interfaces.

    **colSpan/wrapping:** With antd Table, colSpan empty state is handled automatically by `locale.emptyText`.

    **Verify:** `npm run typecheck` passes on the modified file.
