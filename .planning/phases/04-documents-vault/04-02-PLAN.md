## Draft Generation Flow

Per D-05 through D-08, D-18:
1. Specialist selects template from approved/published list
2. System loads intake answers for the request
3. Specialist fills variable values form (pre-filled from intake answers where keys match)
4. System validates required variables — missing required blocks generation
5. System performs placeholder replacement: {{variable_key}} -> variable_value
6. Creates DocumentVersion record with:
   - templateId, templateVersion snapshot
   - inputSnapshot: { variables: {...}, intakeAnswers: {...} }
   - generatedContent
   - status: draft
7. Creates VaultFile for the draft with kind='generated_draft'
8. Records audit event

## Document Version Schema

Per D-09, D-10, D-18:
- DocumentVersion model: first-class immutable records
- Fields: templateId, templateVersion, status (draft/submitted_for_review/final)
- inputSnapshot: JSON with variables and intake answer copy
- generatedContent: the text after placeholder replacement
- relation to Document model (for grouping)

## Vault Service Requirements

Per D-13, D-14, D-15, D-17:
- Extend VaultFile metadata for file kind, version, source, storage key
- RBAC check on every vault access
- Audit events for all file actions
- Never expose raw storageKey as URL — signed URL abstraction (stubbed OK)

## Submit for Review Flow

Per D-11:
- Must reference specific document version
- Must move workflow through backend state machine only
- transitionRequestStatus from in_progress -> pending_review
- Audit event with document version reference

    **Placeholder replacement algorithm:**
    ```typescript
    function replacePlaceholders(content: string, variables: Record<string, unknown>): string {
      return content.replace(/\{\{(\w+)\}\}/g, (match, key) => {
        const value = variables[key];
        if (value === undefined || value === null) return '';
        if (typeof value === 'object') return JSON.stringify(value);
        return String(value);
      });
    }
    ```

    **Variable validation:**
    ```typescript
    function validateRequiredVariables(schema: TemplateVariable[], variables: Record<string, unknown>): string[] {
      return schema
        .filter(v => v.required && (variables[v.key] === undefined || variables[v.key] === null || variables[v.key] === ''))
        .map(v => v.key);
    }
    ```
