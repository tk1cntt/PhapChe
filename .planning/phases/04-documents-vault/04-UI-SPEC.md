## Component Inventory

### Admin Template Management (/admin/templates)

#### Template List Page
- **PageHeader**: title="Quản lý mẫu tài liệu", description="Tạo, chỉnh sửa và xuất bản mẫu tài liệu pháp lý"
- **Table columns**: Tên mẫu, Phiên bản, Trạng thái, Loại vụ việc, Ngày tạo, Thao tác
- **Status Badge tones**:
  - Nháp (draft) → tone="neutral"
  - Đã duyệt (approved) → tone="info"
  - Đã xuất bản (published) → tone="accent"
  - Không còn sử dụng (deprecated) → tone="outline"
- **Row actions**: Xem/Sửa, Tạo phiên bản mới, Xem lịch sử
- **Empty state**: Card with centered text "Chưa có mẫu tài liệu nào" + primary button "Tạo mẫu đầu tiên"

#### Template Detail/Edit Page
- **PageHeader**: title="Chi tiết mẫu tài liệu", back link to list
- **Form fields**: Tên mẫu, Mô tả, Loại vụ việc (select), Trạng thái (select, disabled if published)
- **Variable schema editor**: Dynamic list with add/remove
  - Each variable: key (text), label (text), type (select: text/number/date/select), required (checkbox)
- **Version history table**: columns = Phiên bản, Ngày tạo, Trạng thái
- **Actions**: Lưu (primary), Hủy (secondary/destructive)
- **Validation error**: red-50 background, text-[14px] red-600 text

#### Template Creation Form
- **Modal or dedicated page**: Create new template from scratch
- **Required fields validation**: Block submit if missing, show inline errors
- **Template body editor**: textarea for template content with variable placeholders `{{variableKey}}`

### Specialist Workbench (/specialist/requests/[requestId])

#### Extend existing page with:

##### Document Draft Section (GenerateDraftForm)
- **From existing** `src/app/specialist/requests/[requestId]/components/generate-draft-form.tsx`
- **Template selection**: Select dropdown with template label + version
- **Variable inputs**: text/number/date based on variable type
- **Pre-fill**: Variables auto-populated from intake answers (existing pattern)
- **Submit**: "Tạo bản nháp" (primary button)
- **Error**: "Vui lòng chọn một mẫu tài liệu." or validation messages
- **Success**: "Đã tạo bản nháp thành công!" + green-50 banner
- **Empty state**: Card "Chưa có mẫu tài liệu nào phù hợp cho loại vụ việc này."

##### Document Versions Section (DocumentVersionsList)
- **From existing** `src/app/specialist/requests/[requestId]/components/document-versions.tsx`
- **Version cards**: template label + version badge, status badge, timestamp
- **Status Badge tones** (D-10):
  - Nháp (draft) → tone="neutral"
  - Đang chờ kiểm tra (submitted_for_review) → tone="warning"
  - Hoàn thành (final) → tone="accent"
- **Version actions**:
  - Xem nội dung: expand/collapse content preview
  - Gửi kiểm tra: primary button (only if draft status)
- **Success message**: "Đã gửi bản nháp để kiểm tra thành công!"

##### Vault Files Section (VaultFilesList)
- **From existing** `src/app/specialist/requests/[requestId]/components/vault-files.tsx`
- **File kind labels**:
  - Tải lên (intake_upload) → tone="info"
  - Bản nháp (generated_draft) → tone="accent"
  - Phản hồi kiểm tra (review_feedback) → tone="warning"
  - Tài liệu hoàn thành (final_document) → tone="accent"
- **File display**: icon + filename + timestamp + kind badge
- **No download links in Phase 04** (D-17): display metadata only
- **Empty state**: Card "Chưa có tệp nào trong kho lưu trữ."

## Interaction States

- **Default**: white background, #E2E8F0 border
- **Focus**: 2px ring #0F766E with 2px offset
- **Error**: red-50 background, red-600 border/text

### Button States
- **Primary**: #0F766E bg, white text, hover teal-800
- **Secondary**: white bg, border #CBD5E1, hover bg #F8FAFC
- **Destructive**: #DC2626 bg, white text, hover red-700
- **Disabled**: opacity-60

### Loading States
- Button text: "Đang tạo..." during draft generation
- Button text: "Đang gửi..." during submit for review
- Disabled state on submit button during async operation
