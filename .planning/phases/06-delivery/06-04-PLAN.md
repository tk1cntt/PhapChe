## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Customer browser -> customer page | Untrusted route param loads sensitive delivery view. |
| Customer page -> download route | Customer clicks vault-file action link. |
| Specialist form -> server action | Internal user submits status mutation and close reason. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-04-01 | Information Disclosure | Customer page | mitigate | Page calls safe DTO service only; no internal query/imports/fields. |
| T-06-04-02 | Elevation of Privilege | Customer page route param | mitigate | `requireAppSession`, service-side creator/workspace/RBAC checks, `notFound` on forbidden/missing. |
| T-06-04-03 | Tampering | Server actions | mitigate | Actions never update Prisma directly; call delivery service and workflow state machine. |
| T-06-04-04 | Repudiation | Close form | mitigate | Close reason required and passed to workflow transition audit. |
</threat_model>
