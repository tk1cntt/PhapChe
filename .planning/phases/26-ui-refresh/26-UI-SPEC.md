## Color — BRAND REFRESH

### Changes from Phase 14

| Property | Phase 14 (OLD) | Phase 26 (NEW) |
|----------|---------------|----------------|
| Brand name | "GitNexus Legal" | "Pháp Chể" |
| Accent/Primary | #0F766E (teal) | #27AE60 (green) |
| Primary Button | #0F766E | #3498DB (blue) |
| Active nav bg | #F0FDFA | #E8F8EF (green tint) |
| Active nav text | #0F766E | #27AE60 |

### New Theme Tokens

| Role | Value | Usage | Ant Design Token |
|------|-------|-------|------------------|
| Brand Primary (sidebar) | `#27AE60` | Logo, sidebar active state | `colorPrimary` |
| Button Primary | `#3498DB` | Primary action buttons | mapped via Button token |
| Page Background | `#F8FAFC` | Layout background | `colorBgLayout` |
| Container Surface | `#FFFFFF` | Cards, sidebar, nav | `colorBgContainer` |
| Destructive | `#DC2626` | Destructive actions only | `colorError` |
| Text primary | `#0F172A` | Headings, body | `colorText` |
| Text secondary | `#475569` | Descriptions, labels | `colorTextSecondary` |
| Border | `#E2E8F0` | Card/table borders | `colorBorder` |
| Muted background | `#F1F5F9` | Hover, disabled bg | `colorFillTertiary` |

### Accent Rules (NEW)

**Accent `#27AE60` (green) reserved for:**
- Sidebar logo and active navigation indicator
- Success/approved status badges
- Focus ring / outline
- Selection highlight color

**Primary Button `#3498DB` (blue) reserved for:**
- Primary CTA buttons (Form submit, Create actions)
- Call-to-action elements

**Accent is NEVER used for:** destructive actions, regular card borders, secondary buttons, disabled elements.
