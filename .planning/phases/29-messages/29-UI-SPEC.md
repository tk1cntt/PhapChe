## Visual Hierarchy

1. Thread list (left) — navigation to select conversation
2. Chat panel (center) — focal point, main message interaction

---

## Interaction Behaviors

### Thread Selection
- Click thread item → update active state (background #ecfdf9)
- Selected thread loads messages in chat panel
- Info panel updates to show selected thread's case information
- Previous thread loses active state

### Composer
- Input: height 44px, border-radius 8px, placeholder text
- Send button: teal gradient (create-btn style)
- Submit on button click or Enter key
- Disabled state when input is empty

### Floating Chat
- Fixed position: right 22px, bottom 20px
- Always visible on page scroll
- Badge shows unread count
