## Copywriting Contract

All user-facing text is Vietnamese.

### Primary CTA

**Button:** `Xuất báo cáo vận hành`
**Icon:** Chart up-right SVG
**Style:** Teal gradient button (same gradient as Phase 47/48 "Tạo hồ sơ" but teal: `#0b8f86` to `#087970`), 45px height, 18px padding, 8px radius, white text, shadow `0 8px 18px rgba(8, 127, 120, 0.25)`.

### Stat Cards

| Card | Title | Description |
|------|-------|-------------|
| Blue | Hồ sơ đang mở | Tăng X% trong tuần |
| Orange | Sắp quá SLA | Cần xử lý trong 24h |
| Green | Hoàn tất hôm nay | Workflow đã đóng |
| Red | Cảnh báo audit | Yêu cầu rà soát quyền |

### Panel Titles

- Workload panel: `Tổng quan workload`
- Timeline panel: `Timeline audit`

### Workload Item

- Format: `{Avatar} {Name} {Role}` + progress bar + `{N} hồ sơ`
- Role labels: Specialist, Reviewer, Coordinator, Audit (from user role)

### Timeline Item

- Format: `{dot} {bold title}` + description paragraph + relative time
- Time format: relative ("X phút trước", "X giờ trước")
- Dot: teal (#087f78) circle, 18px, 4px border (#d9f8f4)

### Operations Table

| Column | Header |
|--------|--------|
| 1 | Mã hồ sơ |
| 2 | Workspace |
| 3 | Trạng thái |
| 4 | SLA |
| 5 | Ưu tiên |
| 6 | Phụ trách |
| 7 | Thao tác |

**SLA cell:** Shows label (e.g., "Còn 4h", "Trễ 2h", "Đúng hạn", "Chưa có SLA") + percent + color-coded bar.

**Action links:**
- Normal row: `Điều phối →`
- Completed row: `Xem audit →`
- Overdue row: `Xử lý ngay →`

### Toolbar

| Control | Placeholder/Label |
|---------|-------------------|
| Search | Tìm hồ sơ, người phụ trách, workspace... |
| Filter button | Bộ lọc |
| SLA dropdown | SLA |
| Assignee dropdown | Người phụ trách |
| Refresh | (icon only) |
| Export | Export |
| Columns | Cột hiển thị |

### Empty State

- **Heading:** Chưa có hồ sơ nào
- **Body:** Hiện tại chưa có hồ sơ phù hợp với bộ lọc. Thử thay đổi điều kiện lọc hoặc tạo hồ sơ mới.
- **Next step:** Nhấn "Tạo hồ sơ" hoặc xóa bộ lọc để xem tất cả.

### Error State

- **Heading:** Không thể tải dữ liệu vận hành
- **Body:** Đã xảy ra lỗi khi lấy dữ liệu từ máy chủ. Vui lòng thử lại.
- **Retry CTA:** Thử lại

### SLA Fallback (D-11)

When `slaDeadline` and workflow transition data are both absent:
- Label: `Chưa có SLA`
- Level: info (blue)
- Bar: empty (#eaf0f6 background, no fill)

### Floating SLA Warning Button

- Label: `{N} Issue ×` where N = near-SLA or overdue count
- Background: `linear-gradient(#ef4444, #dc2626)`
- Border: `3px solid #facc15`
- Left icon: N in circle with `rgba(255,255,255,0.18)` background

### Destructive Actions

None in this phase. "Xuất báo cáo" is a read/export action, not destructive.

## Component Inventory

### AdminOperationsClient (container)

- **Purpose:** Owns loading/error/retry/filter/refresh state; fetches `/api/admin/operations`
- **States:** loading, error (with retry), empty, data
- **Pattern:** Mirrors `AdminRequestsClient` from Phase 48

### AdminOperationsStats

- **Purpose:** Renders 4 stat cards from DB data
- **Component:** Reuses `AdminStatGrid` + `StatCard` from `src/components/admin/AdminStatGrid`
- **Variants:** blue, orange, green, red (matching template)

### AdminOperationsWorkload

- **Purpose:** Specialist/reviewer workload panel with progress bars
- **Component:** Clone/adapt from `src/components/admin/WorkloadPanel`
- **Matches:** `layout/admin-operations.html` workload grid (190px name / 1fr progress / 70px count)
- **Progress bar fill:** gradient `#0b8f86` to `#22c55e`

### AdminOperationsTimeline

- **Purpose:** Global recent operations/audit timeline
- **Component:** Clone/adapt from `src/components/admin/AuditTimeline`
- **Matches:** Timeline dot teal (#087f78), border (#d9f8f4), 18px dot, 38px left padding
- **Data source:** D-04: global timeline from `AuditEvent` + `WorkflowTransition`

### AdminOperationsToolbar

- **Purpose:** Filter/search toolbar
- **Component:** Reuses `AdminToolbar` from `src/components/admin/AdminToolbar`
- **Extends:** Add SLA dropdown, Assignee dropdown for operations-specific filters (workspace, status, matter type, specialist, reviewer, date range per D-03)

### AdminOperationsTable

- **Purpose:** 7-column operations table with SLA bars
- **Matches:** Template table grid `0.9fr 1.15fr 1fr 1.05fr 0.9fr 0.95fr 1fr`
- **SLA bar:** 8px height, border-radius 999px, background #eaf0f6
- **Status badges:** blue/orange/green/red with dot prefix
- **Priority badges:** red/orange/green with dot prefix
- **Empty row:** "Chưa có hồ sơ nào phù hợp với bộ lọc"
- **Paging:** Uses `Paging` from `src/components/ui/Paging` (Phase 48 pattern)

### FloatingSLAButton

- **Purpose:** Fixed-position red warning button
- **Matches:** Template `.floating-chat` — `border-radius: 999px`, gradient `#ef4444` to `#dc2626`, yellow border `3px solid #facc15`
- **Shows when:** Near-SLA or overdue count > 0
