## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-50-01 | Tampering | API params | mitigate | Parse and validate all numeric params with defaults; reject negative values |
| T-50-02 | Information Disclosure | API response | mitigate | Only expose actor email, workspace name, action, targetType, targetId, correlationId, metadataSummary - no raw legal content |
| T-50-03 | Denial of Service | API pagination | mitigate | Enforce max pageSize of 100; requireAppSession prevents anonymous abuse |
| T-50-04 | Authorization | API access | mitigate | requireAppSession + ADMIN_ROLES check; return 403 for non-admins |
</threat_model>
