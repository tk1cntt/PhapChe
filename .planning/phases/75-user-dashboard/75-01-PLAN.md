# Plan 75-01: User Dashboard Enhancement

## Objective

Enhance existing User Dashboard với clickable stat cards, floating chat badge, loading/error/empty states, pagination, CSS matching, và i18n verification.

## Background

Dashboard page tại `src/app/[locale]/dashboard/page.tsx` đã có server-side Prisma queries lấy real data. Component `DashboardClient.tsx` nhận props từ server và render 6 sub-components. Cần enhance để match SPEC requirements.

## Tasks

### Task 1: Make Stat Cards Clickable

**Read first:**
- `src/components/dashboard/StatCard.tsx`
- `src/components/dashboard/DashboardClient.tsx`
- `src/components/dashboard/dashboard.css`

**Action:**
1. Update `StatCard` interface to accept optional `href` prop
2. Wrap card content in Next.js `<Link>` when `href` provided
3. Update `StatsCardGrid` to pass href props:
   - `totalRequests` → `/cases`
   - `inProgress` → `/cases?status=in_progress`
   - `completed` → `/cases?status=completed`
   - `vaultDocs` → `/vault`
4. Add `cursor-pointer` CSS class for hover effect

**Acceptance criteria:**
- [ ] StatCard accepts `href` prop
- [ ] When href provided, card renders as clickable link
- [ ] Each stat card links to correct destination
- [ ] Hover effect visible on stat cards

### Task 2: Floating Chat Badge với Dynamic Unread Count

**Read first:**
- `src/components/dashboard/DashboardClient.tsx`
- `src/app/api/messages/route.ts` (check if exists)

**Action:**
1. Create `/api/messages/unread-count/route.ts` endpoint:
   - Query: `prisma.message.count({ where: { recipientId: userId, readAt: null } })`
   - Return: `{ unreadCount: number }`
2. Update `DashboardClient.tsx`:
   - Add `use client` state for `unreadCount`
   - Add `useEffect` to fetch from `/api/messages/unread-count` on mount
   - Update floating chat button to show badge when `unreadCount > 0`
3. Update floating chat button with badge styles

**Acceptance criteria:**
- [ ] `/api/messages/unread-count` returns correct count
- [ ] Floating chat shows badge when unreadCount > 0
- [ ] Badge displays unread count number
- [ ] No badge when unreadCount = 0
