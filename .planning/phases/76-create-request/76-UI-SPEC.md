## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, button icon gaps |
| md | 16px | Default element spacing, form field gaps |
| lg | 24px | Section padding, card internal padding |
| xl | 32px | Layout gaps, wizard step content padding |
| 2xl | 48px | Major section breaks, page header margin |
| 3xl | 64px | Page-level spacing (not used in wizard) |

**Exceptions:**
- Touch targets: 44px minimum (domain cards, service type cards, buttons)
- Progress bar height: 8px (desktop), 6px (mobile)
- Input height: 44px (consistent with Phase 74)
- Button height: 44px (consistent with Phase 74)

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 | 1.5 |
| Label | 13px | 600 | 1.4 |
| Heading | 20px | 800 | 1.2 |
| Display | 31px | 800 | 1.1 |

**Usage:**
- **Body (14px/400):** Question text, descriptions, file names, answer values
- **Label (13px/600):** Form field labels, step names, button text, badges
- **Heading (20px/800):** Section titles within steps ("Chọn lĩnh vực", "Tải lên tài liệu")
- **Display (31px/800):** Page title "Tạo yêu cầu mới" (only on desktop, 24px on mobile)

**Line heights:**
- Body text: 1.5 (readability for question descriptions)
- Headings: 1.2 (tight for visual hierarchy)
- Labels: 1.4 (compact for form density)

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#f8fafc` (--bg) | Page background, wizard container |
| Secondary (30%) | `#ffffff` (--panel) | Cards, step content panels, modals |
| Accent (10%) | `#087f78` (--teal) | Primary buttons, active step indicator, selected domain card border, progress bar fill, links |
| Destructive | `#ef4444` (--red) | Remove file button, delete draft confirmation |

**Accent reserved for:**
1. Primary CTA buttons ("Tiếp tục", "Gửi yêu cầu")
2. Active step in progress indicator (filled circle + label)
3. Selected domain card border (3px solid teal)
4. Selected service type card border (3px solid teal)
5. Progress bar fill (completed steps)
6. "Chỉnh sửa" links in review step
7. Focus ring on inputs (2px teal, 0.1 opacity)

**Secondary colors:**
- `#2563eb` (--blue): Priority badge "Urgent"
- `#f97316` (--orange): Priority badge "Normal" (default)
- `#64748b` (--muted): Disabled buttons, placeholder text, step connector lines

- **Layout:** Horizontal, 5 steps evenly spaced

## Interaction Specifications

### Step Transitions
- **Animation:** Fade out current step (200ms), fade in next step (200ms)
- **Easing:** `cubic-bezier(0.4, 0, 0.2, 1)`
- **Total duration:** 400ms (< 500ms requirement)
- **Progress bar:** Smooth fill transition (300ms)

### Form Validation
- **Trigger:** On blur + on submit (Phase 74 pattern)
- **Error display:**
  - Inline below field (13px, 500 weight, `#ef4444`)
  - Field border: 2px solid `#ef4444`
  - Error icon: 16px, `#ef4444` (alert circle)
- **Scroll behavior:** On submit error, scroll to first error (smooth, 300ms)

### File Upload
- **Drag-over:** Visual feedback (border color, background change)
- **Upload progress:** Progress bar in file card (smooth animation)
- **Error:** Toast notification (react-hot-toast), file card shows error state (red border)
- **Remove:** Instant removal with fade-out (200ms)

### Draft Auto-Save
- **Trigger:** Before step transition (4 saves: 1→2, 2→3, 3→4, 4→5)
- **Visual feedback:**
  - Saving: "Đang lưu..." text (13px, 500 weight, `#64748b`) + spinner (16px)
  - Saved: "Đã lưu" text (13px, 500 weight, `#087f78`) + checkmark (2s, then fade)
  - Error: Toast notification, wizard continues (non-blocking)

### Success State
- **Display:** Full-screen overlay (semi-transparent `rgba(0,0,0,0.5)`)
- **Modal:**
  - Width: 480px
  - Background: `#ffffff`
  - Border radius: 15px
  - Padding: 48px
  - Box shadow: `0 25px 50px rgba(0, 0, 0, 0.25)`
- **Content:**
  - Icon: 64px, `#10b981` (success checkmark)
  - Heading: 24px, 800 weight, `#0f172a`
  - Subtext: 16px, 400 weight, `#64748b`
  - Button: Primary CTA style
  - Spacing: 24px between elements
- **Auto-redirect:** 2s countdown, button visible immediately

## Responsive Breakpoints

### Desktop (≥1024px)
- Wizard max-width: 1200px
- Domain grid: 3 columns
- Step 5 layout: 70/30 split

### Tablet (768-1023px)
- Wizard max-width: 100%
- Domain grid: 2 columns
- Step 5 layout: Single column (sidebar below summary)
- Compact progress indicator (step numbers only)

### Mobile (<768px)
- Wizard max-width: 100%
- Domain grid: 1 column
- Step 5 layout: Single column
- Progress bar + text only
- Reduced padding (24px horizontal, 32px vertical)
- Page title: 24px (instead of 31px)

### Small Mobile (<640px)
- All buttons full-width (100%)
- File upload drop zone: 160px height
- Reduced card heights (domain: 100px, service: 72px)

## Accessibility

### Keyboard Navigation
- **Tab order:** Logical flow (progress → content → buttons)
- **Focus ring:** 2px solid `#087f78`, 0.1 opacity, 2px offset
- **Enter/Space:** Activate buttons, select cards
- **Escape:** Close modals, cancel actions

### ARIA Labels
- Progress indicator: `role="progressbar"`, `aria-valuenow`, `aria-valuemin`, `aria-valuemax`
- Step buttons: `aria-current="step"` (active), `aria-label="Step {n}: {name}"`
- Domain cards: `role="button"`, `aria-pressed="true/false"`
- File upload: `role="button"`, `aria-label="Upload files"`
- Error messages: `role="alert"`, `aria-live="polite"`

### Color Contrast
- All text meets WCAG AA (4.5:1 ratio)
- Teal accent on white: 4.7:1 ✓
- Muted text (`#64748b`) on white: 4.6:1 ✓

### Touch Targets
- Minimum 44px × 44px for all interactive elements
- Domain cards: 120px height ✓
- Service cards: 80px height ✓
- Buttons: 44px height ✓
