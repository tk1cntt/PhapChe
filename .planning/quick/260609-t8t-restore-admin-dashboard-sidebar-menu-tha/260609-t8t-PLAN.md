# Quick Plan: restore-admin-sidebar

## Task 1: Fix [locale]/admin/layout.tsx to render sidebar

**files:**
- `src/app/[locale]/admin/layout.tsx`
- `src/app/admin/layout.tsx`

**action:**
The `[locale]/admin/layout.tsx` only provides NextIntlClientProvider but doesn't render the sidebar. We need to make it either:
1. Import and render the admin layout (as a server component wrapper), OR
2. Move the sidebar code into the [locale]/admin layout

**Solution:** Update `[locale]/admin/layout.tsx` to import and render the admin layout with sidebar, wrapping children in the NextIntlClientProvider.

```tsx
// src/app/[locale]/admin/layout.tsx
import { NextIntlClientProvider } from 'next-intl';
import { getMessages } from 'next-intl/server';
import AdminLayout from '@/app/admin/layout';

export default async function AdminLocaleLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  const messages = await getMessages();
  return (
    <NextIntlClientProvider messages={messages}>
      <AdminLayout>{children}</AdminLayout>
    </NextIntlClientProvider>
  );
}
```

**verify:**
- `/zh/admin/users` renders with sidebar menu visible
- Sidebar shows all nav items (users, workspaces, requests, ops, audit, vault)

**done:**
- [locale]/admin routes have sidebar
