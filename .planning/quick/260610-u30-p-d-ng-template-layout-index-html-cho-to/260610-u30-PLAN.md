# Quick Task: Áp dụng template layout/index.html cho toàn bộ các trang admin

## Goal
Chuyển đổi hoàn toàn giao diện admin từ Ant Design components sang HTML/CSS thuần theo template `layout/index.html`.

## Template Design Key Points

### Colors
- `--bg: #f8fafc` - Background
- `--panel: #ffffff` - Cards/Panels
- `--text: #152238` - Text
- `--muted: #64748b` - Muted text
- `--border: #dfe7f1` - Borders
- `--teal: #087f78` - Primary accent
- `--blue: #2563eb` - Primary button
- `--purple: #6d28d9` - Secondary accent
- `--orange: #f97316` - Warning/pending
- `--red: #ef4444` - Danger
