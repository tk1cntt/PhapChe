## Fix Submit Validation - Required Answers Missing

### Problem
CreateRequestForm submit fails with 400 "Required answers are missing" because:
1. CreateRequestForm doesn't have a step to input intake questions
2. When submit is called, `IntakeSubmission.answers = {}`
3. `validateAnswers()` fails because required fields are empty

### Solution
Modified `validateAnswers()` to skip validation when matterType has no required questions.

### Changes Made
1. `src/lib/intake/intake-service.ts` - Added check for `requiredQuestions.length === 0`
2. `e2e/intake-flow.spec.ts` - Updated tests to check for validation errors
